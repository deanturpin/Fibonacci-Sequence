[Fibonacci sequence](https://en.wikipedia.org/wiki/Fibonacci_number) generator
implemented in multiple languages:
- C
- Haskell
- Node.js
- Bash
- Lua

### Build and run
```bash
$ make clean
rm -f *.o
```

```make``` builds and runs everything.
```bash
$ make
```

### Dependencies
```
clang
nodejs
ghc
lua5.3
```
